# Pleasant-Walk-Finder
A simple software project that finds a walk with preferred weather conditions, making use of Visual Crossing's Weather API.


## Requirements
- Python 3.10+
- requests
- geopy
- a Visual Crossing Weather API key


## Setup

### 1. Download and Install Python 3.10+
`python.org/downloads/`

### 2. Install dependencies:
```bash
pip install -r requirements.txt
```

### 3. API Key
This project uses the Visual Crossing Weather API.  
`visualcrossing.com/weather-api/`  
Replace the placeholder text in `API_Key.txt` with your API key.
